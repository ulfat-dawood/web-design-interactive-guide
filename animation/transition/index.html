Transition

  • The value of some CSS properties might change for various reasons:
    • Changing the color, size, opacity..etc of an element show emphasis when it's is being hovered over or selected.
    • Changing the element properties for different screen size.
    • Changing the visibility of some elements from hidden to visible.
    • And many other changes are added for other reasons.
  • The change in properties of the elements can be triggered by either:

    1. Pseudo Classes:
      • :hover :active :focus :checked :disabled :enabled :valid :invalid

    2. Media Queries:
      • @meida rule.

    3. JavaScript event listners:
      • Add/remove classes to/from the element.
      • Change the value of element properties.

When change is triggered, the transition from a value to another (e.g. from width:200px to width:300px)
can be animated smoothly over a given duration,
by specifying the transition property.

The transition property defines:
  1. The property on which the transition is applied.
  2. Duration of the animation.
  3. Timing function of the animation.
  4. Delay of the animation.

Box Example

Hover over the box to trigger the animated transition.

width will change from 200px to 300px
  • transition-property: width;
  • Defines the propery on which the transition animation is applied.
  • transition-duration: 1s;
  • The transition takes 1 second to animate the change.
  • transition-timing-function:
    linear | ease | ease-in | ease-out | ease-in-out;
  • Defines the speed curve of the transition animation.
  • transition-delay: 0.5s
  • The animation starts 0.5 second after the transition is trigggered .

shorthand

  • transition: transition-property transition-duration transition-timing-function transition-delay

    • transition: width 1s linear 0.5s;

Box Example Code

Hover over the box to trigger the animated transition.

width will change from 200px to 300px

HTML

<div class="box">
     width will change from 200px to 300px
</div>

Styling Box

.box{
     width: 200px;
     transition: width 1s linear 0.5s;
}

Trigger Transition

.box:hover{
     width: 300px;
}
