Keyframes

@keyframes rule

  • Allows to define the keyframes by specifying CSS styles
    to each  keyframes-selector.

@keyframes rule syntax

@keyframes [animation-name] {
    keyframes-selector {css-styles;}
    keyframes-selector {css-styles;}
    keyframes-selector {css-styles;}
}

keyframes-selector

  1. to and from key words:
    allow to define only two keyframes (the start and end of the animation)

  2. percentage-based keyframes-selector:
    allow to define more than two keyframes, where:
    • 0% is equivalent to from (start of the animation).
    • 100% is equivalent to to (end of the animation).

animation-name

[animation-name] is then assigned to the element that will be animated, using the animation property.



.my-element{
   animation: my-animation 2s ease;
}


@keyframes my-animation {
    from{ color:red; }
    to{ color:white; }

}

Examples

Hover over each section below code snippets to play the animation.

Sliding Animation

@keyframes slide-anim {
    from{ left:0%; }
    to{ left:100%; }

}

Spinning Animation

@keyframes spin-anim {
    0%{ transform: rotate(0deg);}
    100%{ transform: rotate(359deg);}

}

Sliding while Changing Color

@keyframes rainbow-anim {
    0%{ left:0%; background:#d4abab}
    25%{background: #abacd4;}
    50%{background: #abd4d2;}
    75%{background: #abd4ab;}
    100%{ left: 100%; transform: translateX(-100%); background:#d4d3ab}

}

Fade in/out

@keyframes fade-anim {
    0%{ opacity: 0; }
    50%{opacity : 1}
    100%{ opacity: 0;}

}
