Animation

The @Keyframes rule defines: Keyframes for each [animation-name]

  • The animation property accepts the [animation-name] and defines further properties for the animation, such as:
    • duration
    • delay
    • iteration-count
    • direction
    • timing-function
    • fill-mode

  • The animation applies the animation to the element/s it is assigned to ..

Animation Properties

Animation Name

  • animation-name: [animation-name];

    • Accepts the animation name that is declared in the @keyframes rule
    • animation-name: my-anim;

Animation Duration

  • animation-duration: [time];

    • Time is specified in either seconds s or milliseconds ms
    • animation-duration: 2s;

Animation Delay

  • animation-delay: [time];

    • Specifies the delay after which the animation is played.
    • Time is specified in either seconds s or milliseconds ms
    • animation-delay: .5s;

Animation Repetition

  • animation-iteration-count:
    [number-of-iterations] | infinite;

    • Specifies the number of times animation is repeated.
    • The keyword infinite denotes the animation is played forever.

Animation Direction

  • animation-direction:
    normal | reverse | alternate | alternate-reverse;

    • normal: the animation plays in the sequence of the normally defined keyframes order.
      • from or 0% is the first keyframe.
      • to or 100% is the last keyframe.
    • reverse: reverses the order of keyframes (the animation is play in revers)
      • to or 100% is the first keyframe.
      • from or 0% is the last keyframe.
    • alternate:
      • its affect is visible when the animation is repeated more than once.
      • odd number iterations are played normally.
      • even number iterations are played in reverse.
    • alternate-reverse:
      • its affect is visible when the animation is repeated more than once.
      • first and all odd number iterations are played in reverse.
      • even number iterations are played normally.

Animation Acceleration

  • animation-timing-function: linear | ease | ease-in |
    ease-out | ease-in-out | cubic-bezier(n,n,n,n);

    • Timing functions: specify the speed curve (speed of the animation over time).
    • CSS provides a set of predefined timing functions.
    • cubic-bezier(n,n,n,n): allows to define custom timing functions.

Animation Fill Mode

  • animation-fill-mode: forwards | backwards | both;

  • Defines the state of the animated element before and after the animation.
  • forwards: After the animation ends, the animated element retains the styles set by the last keyframe.
  • backwards: Before the animation starts (during the animation delay time), the styles set by the first keyframe is applied to the element.
  • both: Sets the fill mode to both forwards and backwards.

Pausing/Playing an Animation

  • animation-play-state: paused | running;

  • Defines the initial state of the animation to either be paused or running.
  • Allows to pause or play the animation at any point of time.
    • paused: pauses a running animation at the exact state when this property is triggered
    • running: resume the animation from the last paused state.
