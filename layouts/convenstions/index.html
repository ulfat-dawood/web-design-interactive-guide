Layout Conventions

Why Follow Conventions?

  • Why follow web design standards like:
    • using <h1> - <h6> and not just simply changing font-size property of a <div>.
    • Placing logo and navigation links on the top header.

  • Because..
    • Conventions make websites feel familiar from site to site, making it easier for any user to navigate through the webpage.
    • These conventions help search engines and web crawlers to easily understand webpage content and access every page of a website across the Internet.

best practices

Navigation Links

Always enclose any section that includes some means for navigation within the <nav> tag.

Lists

  • Navigation links, menus, contacts or any other list of items should alway be contained within either:

    • An unordered list <ul>, or
    • An ordered list <ol>.

  • Where each item in the list is wrapped within:

    • A list item <li> tag.

Containers

To give a webpage a unified look and better readability, it's recommended to wrap all the main sections of the page within containers that have even/similar margin to center the content of the page and give all sections a consistent margin.

fonts

  • Use less font families, max 2 or 3, for headers and body text.
  • But how to add variation and contrast to different text?
  • You cna play around with:
    1. Colors: hue, saturation or lightness.
    2. Text style: weight, size or capitalization.
    3. Whitespace: letter spacing, line-height or margin.

Keep It Simple Stupid

Always remember, less is more.
While exploring all CSS properties is important, using the stylistic features of CSS in moderation is a key to better web design.
