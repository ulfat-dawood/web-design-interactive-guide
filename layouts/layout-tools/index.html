Layout Conventions

Why Follow Conventions?

  • Why follow web design standards like:
    • using <h1> - <h6> and not just simply changing font-size property of a <div>.
    • Placing logo and navigation links on the top header.

  • Because..
    • Conventions make websites feel familiar from site to site, making it easier for any user to navigate through the webpage.
    • These conventions help search engines and web crawlers to easily understand webpage content and access every page of a website across the Internet.

Containers

Wrapping sections with a container to give the page layout consistent look by adding even/similar margin to all sections on the page.
