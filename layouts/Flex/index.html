FlexBox

When to Use Flexbox

  • Flexbox is used when building layouts in one dimension (either colums and rows).
  • Small-scale layouts (row of cards, colomn of menu items in a sidebar).

Why Flexbox?

  • Building layouts with more FLEXibility and stability than floats, inline-blocks and absolute positioning.
  • Easily..
    • Setting equal or varying width/height of elements
    • Managing space between/around elements.
  • Building responsive webpages, by easily changing the direction and/or order of elements.

Flexbox Properties

There are separate properties for:

  • The flex container
  • The flex items

Flex Container

display: flex;

Direction and Wrap

  • Default direction is ROW.
  • WRAP: allow flex items that don't fit to wrap to the next line, instead of trying to fit in one line.
  • flex-direction: row | column | row-reverse | column-reverse;
  • flex-wrap: nowrap | wrap | wrap-reverse;

Sorthand :

  • flex-flow: flex-direction flex-wrap;

Items Alignment

  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
    • Defines items alignment along the set direction
    • sets the horizontal alignment when direction is row).
