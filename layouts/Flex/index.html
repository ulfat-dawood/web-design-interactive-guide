FlexBox

When to Use Flexbox

  • Flexbox is used when building layouts in one dimension (either columns and rows).
  • Small-scale layouts (row of cards, column of menu items in a sidebar).

Why Flexbox?

  • Building layouts with more FLEXibility and stability than floats, inline-blocks and absolute positioning.
  • Easily..
    • Setting equal or varying width/height of elements
    • Managing space between/around elements.
  • Building responsive webpages, by easily changing the direction and/or order of elements.

Flexbox Properties

There are separate properties for:

  • The flex container (parent element).
  • The flex items (children elements).

Flex Container

Define the flex container

display: flex;

Direction and Wrap

  • Default direction is ROW.
  • WRAP: allow flex items that don't fit, to wrap to the next line instead of trying to fit in one line.
  • flex-direction: row | column | row-reverse | column-reverse;
  • flex-wrap: nowrap | wrap | wrap-reverse;

Shorthand :

  • flex-flow: flex-direction flex-wrap;

Items Alignment

  • justify-content: flex-start | flex-end | center |
    space-between | space-around | space-evenly;

    • Defines items alignment along the set direction
    • sets the horizontal alignment when direction is row.

  • align-items: flex-start | flex-end | center |
    stretch | baseline;

    • sets the vertical alignment of all items when direction is row.

Wrapped Lines Alignment

  • align-content: flex-start | flex-end | center |
    space-between | space-around | space-evenly |
    stretch | baseline;

    • Sets the vertical alignment of each line when direction is row.

Flex Items

ALIGNMENT of an Individual Item

  • align-self: flex-start | flex-end | center |
    stretch | baseline ;

    • sets the vertical alignment of one item when direction is row.

SIZE of an Individual Item (fixed or proportional)

  • Set an item's width (when direction is row).
  • Set an item's height (when direction is column).

Fixed :

  • flex-basis: [number][unit];

Proportional :

  • flex-grow: [unitless number];

Shorthand :

  • flex: none | (flex-grow || flex-basis);

ORDER of an Individual Item

  • order: [unitless number];
    • Accepts negative values as well.
