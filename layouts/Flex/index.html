FlexBox

When to use flexbox

  • Flex box is used when building a layouts in one dimension (either colums and rows)

Why flexbox?

  • Building layouts with more FLEXibility and stability than floats, inline-blocks and absolute positioning.
  • Easy to set equal or varying width/height of elements and manage space between/around them.
  • Building responsive webpages, by easily changing the direction and order of elements.
