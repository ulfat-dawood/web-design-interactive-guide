Grid

When to use grid?

  • Grid is used when building a layouts in two dimension (rows and columns at the same time).
  • Large-scale layouts.

Grid Properties

There are separate properties for:

  • The grid-container (parent element).
  • The grid-items (children elements).

The grid-container is divided into rows and columns that contains:

  • grid-cells (containing the grid-items).
  • grid-lines surrounding grid-cells (each line has a number identifier).

Grid Container

Define the Grid Container

  • display: grid;

Number of Rows/Columns and their Size

  • grid-template-columns:
    [col-1-width] [col-2-width]...[col-n-width] ;

  • grid-template-rows:
    [row-1-height] [row-2-height]...[row-n-height] ;

  • grid-template-columns: 50% 100px 50px;
  • grid-template-rows: 1fr 2fr 100px;
  1. First, last row will occupy 100px of the grid-container-height.
  2. Then, first and second rows will fill all the available space.
  • fr forced rows to fill all the available space of the grid-container height.
  • Columns don't fill the entire grid-container's width.

Grid Areas

  • grid-area: one or multiple adjacent grid-cells identified by a given label.
  • Later, a grid-item can be assigned to one the grid-areas to fill that portion of the grid.

  • grid-template-columns: 10% 1fr 20%;
  • grid-template-rows: 10% 1fr 15%;
  • grid-template-areas:
    "header header header"
    "nav main aside"
    "nav footer . "

The period after footer indicate an empty cell.

Gap around grid-lines

column-gap / row-gap specify the GUTTER (the gap around INNER grid-lines only, without the grid-lines on the edge).

  • row-gap: 40px;
  • column-gap: 20px;

Shorthand :

  • gap: row-gap column-gap;

  • gap: 40px 20px;
  • gap: 30px; //sets both row-gap and column-gap to 30px

grid-items Alignment within grid-cells

  • Align all grid-items horizontally within their grid-cells.
    • justify-items: start | end | center | stretch;

  • Align all grid-items vertically within their grid-cells.
    • align-items: start | end | center | stretch;

  • justify-items: center;
  • align-items: start;

Shorthand

  • place-items: align-items justify-items

  • place-items: end end;
  • place-items: end; //sets both to 'end'

grid-cells Alignment within the grid-container

  • Align Columns of grid-cells Horizontally
    • justify-content: start | end | center | stretch | space-around | space-between | space-evenly;

  • The effect of justify-content won't be reflected if columns fill the entire grid-container's width.
justify-content: start; justify-content: center; justify-content: space-between;
  • Align Rows of grid-cells vertically
    • align-content: start | end | center | stretch | space-around | space-between | space-evenly;

  • The effect of align-content won't be reflected if rows fill the entire grid-container's height.

Shorthand

  • place-content:align-content justify-content;

  • place-content:center center;
  • place-content:center; //sets both properties to 'center'

Grid Items

grid-items Location

By default, each grid-item will be located inside on of the grid-cells with respect to their order.

However, this can be changed, by locating the grid-items to a differnt grid-cell/s or to a pre-defined grid-area.

grid-area : one or multiple adjacent grid-cells identified by a given label.


This is the default location of grid-item that is
the first child of grid-container

Locate grid-items by: grid-lines

  • grid-column-start: 1;
  • grid-column-end: 4;
  • grid-row-start: 2;
  • grid-row-end: 3;

Shorthand

  • grid-column: grid-column-start grid-column-end;
  • grid-row: grid-row-start grid-row-end;

Locate grid-items by: grid-area

  • .grid-container{
    grid-template-areas:
    "header header header"
    "nav main aside"
    "nav footer . "
    }
  • .grid-item-1{ grid-area: header; }
  • .grid-item-2{ grid-area: nav; }
  • .grid-item-3{ grid-area: main; }
  • .grid-item-4{ grid-area: aside; }
  • .grid-item-5{ grid-area: footer; }

Alignment of Individual grid-item

  • Align a specified grid-item horizontally within its grid-cells.
    • justify-self: start | end | center | stretch;

  • Align a specified grid-item vertically within its grid-cells.
    • align-self: start | end | center | stretch;

  • justify-items: end;
  • align-items: start;

Shorthand

  • place-self: align-self justify-self

  • place-self: center center;
  • place-self: center; //sets both to 'center'
