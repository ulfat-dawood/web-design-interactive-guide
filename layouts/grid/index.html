Grid

When to use grid?

  • Grid is used when building a layouts in two dimension (rows and columns at the same time).
  • Large-scale layouts.

Why grid?

Grid Properties

There are separate properties for:

  • The grid-container (parent element).
  • The grid-items (children elements).

The grid-container is divided into rows and columns that contains:

  • grid-cells (containing the grid-items).
  • grid-lines surrounding grid-cells (each line has a number identifier).

Grid Container

Define the Grid Container

  • display: grid;

Rows/Columns Number and Size

Grid Areas

A collection of adjacent cells, identified by a label.
Later, a grid item is assigned this label to fill the designated area.

Lines Size

Spesifies the size of the INNER lines (lines between columns and rows not the lines forming the edge of the grid).
