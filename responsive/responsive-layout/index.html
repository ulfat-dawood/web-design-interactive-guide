Responsive Web Design

What is responsive web design?

It is a design where the layout and different UI elements respond to adapt with the change in the media where it is being displayed.

Media Queries

Broadly, responsive design refers to the design that adapts by changing the page layout to fit nicely for different screens dimensions (from large desktops, TV screens to small mobile screens).


Breakpoints

Responsive web design can be achieved by the following: using Media Queries @media rule. then adding different breakpoints for multiple screens dimensions (desktop, tablet, mobile...etc).

Before Jumping to Media Queries

  • First, allow the browser to control the webpage dimensions by providing the following <meta> tag in your HTML document.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

responsive UI elements

Width and Margin

  • Use percentage as the unit when assigning width.
    • Give the element 100% width and percentage based margin.
    • Give the element a percentage based width and auto margin.

Height

  • Using percentage based height is tricky, and would most probably not work.
  • Percentage based width is set relative to the browser window size.
  • Percentage based height on the other hand is not set relative to anything, unless explicitly defined.

  • To make hieght responsive, set it relative to the viewport height using the vh unit.

Images

  • Assign images max-width: 100%; to avoid pixelation when the image width grows.
  • Wrap images within a responsive container.
  • set the images size and position using
    • object-fit
    • object-position
