Media Quesris

    The CSS @meida rule defines media queries, which:
  • Targets different media types and their condition to define different styles as the media on which webpage is display changes.

    Media type:
  • Computer Screens.
  • Printers.
    Media Condition:
  • width and height of the viewport
  • orientation
  • resolution

@media Syntax


  • @media not|only media-type and (media-condition) {
    /* Define styles for differnt selectors here */
    }

      • where:
    • media-type: screen | print | speech | all(default)
    • media-condition: width, height, orientation or resolution

Examples

  • @media only screen and (max-width:768px){…}

    • Targets screens, where width is less than 768px.


  • @media (min-width: 500px){…}

    • Targets all media types, where width is larger than 500px.


  • @media only screen and (orientation: portrait) and (max-width: 500px){…}

    • Targets screens, where orientation is portrait and width is less than 500px.


  • @media (min-width: 300px) and (max-width: 500px){…}

    • Targets all media types, where width is between 300px & 500px.
