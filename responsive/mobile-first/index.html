Mobile First Design

In mobile-first desing:

  1. Layout for small screens is designed first.
  2. Media query breakpoints for larger screens are defined later.
  3. The goal is to scale up the layout and UI elements.
  • This design approach is followed when the website traffic is mostly coming from mobile devices.
  • Example: Amazon.

Scaling up the UI

Large-scale Layout (grid-based)

  • Increase the number of grid columns.
  • Distribuite the grid-areas or grid-items to fill multiple columns.

Small-scale Layout(flex-based)

  • Change the direction of flex-container to row or column.
  • Change the order of flex items as needed.

Margins

  • Be generous with margins.
  • When possible, ncreas the gutter (margin between elements) and the margin around the webpage edges.

Navigation Menus

  • Expand the navigation menus to be visible at all times.

Illustrations

  • Since more space is available, whitespace can be filled with illustrations.
