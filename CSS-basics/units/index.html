Units

Types of units

  • Percentage
  • Absolute
  • Relative

Percentage

  • Percentage based on parent element.
  • Mostly used to set responsive width.

This box takes up 90% of it's parent width.
Change the browser window size to test responsive width

Absolute Units

  • cm - centimeters
  • mm - millimeters
  • in - inches
  • pt - points
  • px - pixels

Pixels (px) are used mostly, for all measures.
Although px are fixed and will appear the same for different viewport, they are relative to the viewing device.

Height, width and font-size of this box will remain the same accros all window size.

Relative Units

Relative to Viewport

Viewport == browser window size


  • vh - view height
  • vw - view width

100vh will fit the entire window height.

Height is set to 50vh = it will fill 1/2 of the window height.

Relative to Font Size

  1. em - (EleMent) Relative to the font-size of the element.
  2. rem - (Root EleMent) Relative to the font-size of the root element.

EM

  • MARGIN/ PADDING/ HEIGHT/ WIDTH
    • Relative to the font-size of current element.

    font-size:10px

    padding: 3em (30px)

    font-size:10px

    padding: 2em (20px)

    font-size:10px

    padding: 1em (10px)


  • FONT-SIZE
    • Relative to the font-size of parent element.

    font-size:10px

    font-size:2em (20px)

    font-size:2em (40px)

REM

  • MARGIN/ PADDING/ HEIGHT/ WIDTH / FONT-size
    • Relative to the font-size of root element (html).

    html{font-size:10px}

    font-size: 2rem (20px)

    font-size: 2rem (20px)
