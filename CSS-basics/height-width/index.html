Width & Height

width and height properties accept the following values:

  • [number] % | px | vh | vw | em | rem;
  • max-content
  • min-content
  • fit-content

box-sizing: border-box | content-box

border-box

width and heigh of an element is set while
including BORDER and PADDING as part of the width and height.

contetn-box

width and heigh of an element is set while
excluding BORDER and PADDING from the width and height.

Width

Use px to set fixed width.

Or use % with width, if you want to allow it to grow and shrink relative to the parent width or browser window width .

Height

Use px to set fixed height.

Or use hv to set the height relative to the viewport height , and to allow it to grow/ shrink relative to it.
