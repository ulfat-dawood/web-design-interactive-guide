This text has an inherit color shadow

  • text-shadow: x -y blur color;

Font

Font Style

  • font-style: normal | italic | oblique;
Italic text

Font Variant

  • font variant: normal | small-caps;

  • small-caps:
    • Converts lowercase letterrs to uppercase.
    • Previous lowercase letters appear smaller.
Small-caps Text Appear Like This

Font Weight

  • font-weight: normal | bold | bolder | lighter | [Numeric keyword];

  • Numeric keyword can be:
    • 100 | 200 | 300 | 400 (normal) | 500 | 600 |
      700 (bold) | 800 | 900
This text is bold

Font Size

  • font-size: [number][unit];
HINT: unit can be px, pt, rem or em.
This text is 50 px large

Line Height

  • line-height: normal(default) | [scale] | [number][unit];

  • defines the 'leading' of text, the white space between each line.
HINT: unit can be % or px ...etc.
Lines are separated by 1.75 of the actual font-size

Font family

  • font-family: ['font-name'] , [fallback-font];
  • fall back fonts are usually generic-family fonts, such as:
    • sans-serif
    • serif
    • cursive
    • fantasy
    • monospace
HINT: generic-family fonts don't need to be qouted.
"Times New Roman" is the font of this text.
serif, is the fallback font.

Font Stretch

  • font-stretch: ultra-condensed | extra-condensed | condensed | semi-condensed | normal | semi-expanded | expanded | extra-expanded | ultra-expanded;

  • Defines how streched each letter will apprear.
  • Only Variable fonts like "League Mono Variable" support font-strech .
  • Not supported by older browsers.

Shorthand

  • font: font-size font-family; (mandatory values)

  • font: font-stretch font-style font-variant font-weight font-size/line-height font-family (optional values)
This text is italic with default font variant, 500 font weight, 1.5 line height, Helvetica font and 30px large.

  • vertical-align: baseline|length|sub|super|top|text-top|middle|bottom|text-bottom;

  • Works only:
    • with (inline) or (inline-block) elements.
    • And if float:none.

  • Aligns (children elemnts only) (not text) vertically, relative to their other siblings (NOT parent).

  • vertical-align property is assigned to each individual children (NOT to the parent)
