CSS Box Model

All HTML elements follow the Box Model, which consist of:

  1. Content: where child elements, text, images are placed.
  2. Padding: adds extra area around the content, before the border.
  3. Border
  4. outline: a line outside the border, used to highlight an element when focused.
  5. Margin: area outside the border.

Color can be specified for content background, border and outline.
Both padding and margin are transparent.

Box Sizing

box-sizing: border-box | content-box

border-box

width and heigh of an element is set while
including BORDER and PADDING as part of the width and height.

content-box

width and heigh of an element is set while
excluding BORDER and PADDING from the width and height.
