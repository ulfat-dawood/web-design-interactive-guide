RGB colors

RGB Color Model: Red Green Blue

CSS function rgb(red , green , blue) accepts 3 parameters:

  1. RED intensity: 0-255 | 0%-100%
  2. Green intensity: 0-255 | 0%-100%
  3. Blue intensity: 0-255 | 0%-100%
rgb(255, 0, 0)
rgb(100%, 0%, 0%)
rgb(0, 255, 0)
rgb(0%, 100%, 0%)
rgb(0, 0, 255)
rgb(0%, 0%, 100%)
rgb(0, 0, 0)
rgb(0%, 0%, 0%)
rgb(255, 255, 255)
rgb(100%, 100%, 100%)

HSL colors

HSL Colo Model: Hue Saturation Lightness

CSS function hsl(hue , saturation , lightness) accepts 3 parameters:

  1. Hue: degree on the color wheel (0-360)
    • RED: 0 | 360
    • GREEN: 120
    • BLUE: 240
  2. Saturation: how saturated or desaturated the hue is (0%-100%)
    • 100% is the full color (full saturation)
    • 0% is a shade of gray (completely desaturated)
  3. Lightness: the brightness of the hue (0%-100%)
    • 0% is black
    • 50% is normal
    • 100% is white

  • Fully saturated red, with normal brightness:
    • hsl(0, 100%, 50%) | hsl(360, 100%, 50%)
  • Fully saturated green, with normal brightness:
    • hsl(120, 100%, 50%)
  • Fully saturated blue, with normal brightness:
    • hsl(240, 100%, 50%)
  • Any hue color, any saturation value with full brightness: (white)
    • hsl([0-360], [0-100%], 100%)
  • Any hue color, any saturation value with no brightness:
    • hsl([0-360], [0-100%], 0%)

Color Transparency

Predefined color keywords

transparent

  • use the keyword transparent for 100% Transparency.

Hexadecimal

Make hexadecimal colors transparent by adding 2 more digit: (00 - FF)

#000000
0% transparent
#00000080
50% transparent
#00000033
80% transparent
#00000000
100% transparent

RGB

rgba() function, has a 4th parameter to specify the alpha value: (0-1)

rgb(0, 0, 0)
0% transparent
rgba(0, 0, 0, 0.5)
50% transparent
rgba(0, 0, 0, 0.2)
80% transparent
rgba(0, 0, 0, 0)
100% transparent

HSL

hsla() function, has a 4th parameter to specify the alpha value: (0-1)

hsl(0, 0%, 0%)
0% transparent
hsla(0, 0%, 0%, 0.5)
50% transparent
hsla(0, 0%, 0%, 0.2)
80% transparent
hsla(0, 0%, 0%, 0)
100% transparent
