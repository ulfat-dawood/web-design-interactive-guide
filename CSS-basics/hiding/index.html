Color Transparency VS Opacity

Color Transparency

Trasparency of colors in color related properties can be tuned by appending the alpha channel to the color value.


    The following values of the color black have 50% transparency

  • HEX: #00000080
  • RGB: rgba(0, 0, 0, 0.5)
  • HSL: hsla(0, 0%, 0%, 0.5)
only border is semi-transparent
only background is semi-transparent
only text is semi-transparent

Opacity

The opacity property sets the opacity of the entir element(border/background/text), all it's content and all its descendant elements.

  • opacity: (0-1);
  • 0 implies that the element is fully transparent

  • 1 implies that the element is fully visible.

visibility:hidden VS display:none

Visibility

  • visibility: visible | hidden;
  • visible sets the opacity of the element to 1 .
    The element is completely visible.

  • hidden sets the opacity to 0 .
    The element is completely transparent, yet it takes up space on the page.

display:none

Hides and removes the element from the document layout.
The element no longer occupies space on the page.

parent{
visibility:visible;
display:block}
