Color Transparency VS Opacity

Color Transparency

Trasparency of colors in color related properties can be tuned by appending the alpha channel to the color value.


    The following values of the color black have 50% transparency

  • HEX: #00000080
  • RGB: rgba(0, 0, 0, 0.5)
  • HSL: hsla(0, 0%, 0%, 0.5)
only border is semi-transparent
only background is semi-transparent
only text is semi-transparent

Opacity

Sets the opacity of the entired element(border/background/text), all it's content and all its descendant elements.

  • opacity: (0-1);

Where 0 implies the element is fully transparent and 1 implies element is fully visible.
