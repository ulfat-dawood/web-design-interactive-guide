Position

    The following CSS properties:
  • left
  • right
  • top
  • bottom
    •
  1. Accept numerical numbers, with any unit defined, as the value.
  2. They define how much an element is PUSHED from the left, right, top or bottom.
  3. These properties though, won't have any affect unless the property position is defined.
  • position: static(default) | relative | absolute | fixed | sticky;

Static

  • The default value.
  • When elements positioned static, the properties
    top bottom left right has no effect.

Relative

  • position: relative; element is positioned relative to itself.

  • Default values for top, bottom, left and right is auto.
    change theses values to reflect their effect.
This is positioned relative to itself
  • top: 10px; pushed 10px from top.
  • left: 30px; pushed 30px from left.

Absolute

  • position: absolute; element is positioned relative to the closest (relative/absolute/sticky/fixed) positioned ancestor.
    If none found (i.e. all ancestors are static), it will be positioned relative to <body>.

  • Set the position of the grey elements below to absolute,
    to observe how it will be positioned relative to <body>.
