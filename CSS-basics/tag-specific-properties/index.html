Media Elements

object-fit

  • Defines the height and width of a media element with respect to its parent's.
  • Works with images, videos and other media formats.

  • object-fit: fill | contain | cover | scale-down | none;

change the value of object-fit of the following image:

  • For object-fit property to work, <img/> must have defined height and width.

object-position

  • Changes the position of <img/> , when object-fit property
    is set to either cover or none.
