Stacking

  • To specify the stacking order of elements, define the z-index property, which accepts both positive and negative number values.

  • In order for z-index to work, element must be positioned
    either relative, absolute, fixed or sticky.

  • Default value is z-index:auto;

    Observe how positive, negative, zero and auto values reflect of elements z-index stacking


  • #red{z-index:1}
  • #grey{z-index:auto}
  • #pink{z-index:0}
  • #purple{z-index:-1}
