Overflow

The property overflow is assigned to the parent element to define what should happened when its content does not fit within its fixed boundaries.


  • overflow: visible(default) | hidden | scroll | auto;

    For overflow to have an affect, the parent element must have:
  • A fixed height, that does not grow indefinitely with the content.

The overflow is set to scroll for both elements, BUT..

This element has a fixed height.
The heigh does not grow to fit the content.
This element does not have a fixed height.
The height grows to fit the content.

overflow: visible

This element has a fixed height.
The heigh does not grow to fit the content.

overflow: scroll

On some browsers, the scroll bar will be visible even if contenet isn't overflowing.

This element has a fixed height.
The heigh does not grow to fit the content.

overflow: auto

The scroll bar will be visible only if the content is overflowing.

This element has a fixed height.
The heigh does not grow to fit the content.
