CSS Functions

Functions take one or multiple values as parameters, these function are then passed as a value for CSS properties.

below is list of all these functions

  • url()
  • var()
  • calc()
  • attr()
  • cubic-bezier()
  • Color models
  • Color Gradients

url()

url() function accepts relative/absolute URL for a resource such as an image, font, stylesheet or a filter.

Example

  • background-image: url("//http://placekitten.com/200/300");
  • cursor: url(./mycursor.png);
  • @import url("../style/mobile-media.css");

var()

var() function us used to use the value of an already declared variable.
Learn more about variables.

calc()

calc() function allows to make calculation, even for values of different units.

Example:

.child-box{ width:100%; }
.child-box{ width: calc(100% - 3rem); }

Color Gradients

  • Gradient functions allow to generate smooth transitions between two or more specified colors rather than solid colors.

  • Linear Gradients:

    • linear-gradient(#e4afcb , #e2c58b ,#7edbdc );
    • repeating-linear-gradient( #e4afcb , #e2c58b 10%,#7edbdc 20% );

  • Radial Gradients:

    • radial-gradient( #e4afcb , #e2c58b ,#7edbdc )
    • repeating-radial-gradient( #e4afcb , #e2c58b 20%,#7edbdc 30% );
