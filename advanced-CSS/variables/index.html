CSS Variables

  • Variables in CSS allows developers to assign a value to a variable then reuse the variable thoroughout the entire application.
  • Variables help provide better maintainability, readability and consistency.

  • Variables in CSS are defined within a scope. Variables can then be accessed inside that scope only.

Global scope

  • :root selector defines the global scope (highest-level parent).
  • To be able to access a variable from any scope, it must be defined inside the root selector.
    :root{...}
  • For example, theme colors that will be reused in the entire document must be defined in :root{...}

Local Scope

HTML parent/child structure of elements defines the local scope.

  • All children can access a variable defined for the parent element.
  • A parent element can't access value of a variable defined for one of its children elements.
  • Siblings can't access values of a variable defined for any of their other siblings.
