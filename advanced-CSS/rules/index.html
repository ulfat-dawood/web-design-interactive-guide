CSS Rules

Normally, CSS rules consist of a selector followed by a set of decelerations that define styles for that selector. Yet, CSS rules can do more. bellow a list of other CSS rules:

  • @media
  • @keyframes
  • @import
  • @font-face
  • @charset

@font-face Rule

  • The @font-face rule allows to include fonts other than web save-fonts.
  • @font-face rule should be added to the stylesheet before any styles.

  • @font-face {
      font-family: myFont;
      src: url(sansation_light.woff);
    }

  • .myClass {
      font-family: myFont;
    }
