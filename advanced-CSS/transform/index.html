Transform

The transform property allows to scale, move, rotate and skew elements.

Traslate

Moving an element from its original position

  • transform: translateX(n)

  • where (n) is a numerical value with a unit, to define the distance by which the element is moving along the x-axis.
BOX
  • transform: translateY(n)

  • Defines the distance by which the element is moving along the y-axis.
BOX
  • transform: translate(x,y)

  • Defines the distance by which the element is moving along the x-axis and y-axis.
BOX

Rotate

  • transform: rotate(angle)

  • (angle) is the degree by which the element rotates clockwise.
BOX
  • Element rotates counter-clockwise, when degree is negative.
BOX

Scale

  • transform: scaleX(n)

  • where (n) is a unit numerical scale value, by which, the element size is increased/decreased from its original size by multiplying (n) to the original height and width of the element.

  • scaleX(n) increases/decreases the width of the element.
BOX
  • scaleX(-n) negative values for (n), flips the element horizontally.
Box
  • transform: scaleY(n)

  • scaleY(n) increases/decreases the height of the element.
BOX
  • scaleY(-n) negative values for (n), flips the element vertically.
Box
  • transform: scale(x,y)

  • scale(x,y) increases/decreases the width/height of the element.
BOX

Shorthand :

  • transform: scale(xy)
  • transform: scale(1.5)

  • The element's both width and height are scaled by 1.5 .

skew

  • transform: skewX(angle)

  • (angle) is the degree by which the element is skewed along the x-axis.
BOX
  • transform: skewY(n)

  • Defines the degree by which the element is skewed along the y-axis.
BOX
  • transform: skew(x,y)

  • Defines the degree by which the element is skewed along the x-axis and y-axis.
BOX

Shorthand :

  • transform: skew(xy)
  • transform: skew(20deg)

  • The element is skewed along both x-axis and y-axis by 20deg .
