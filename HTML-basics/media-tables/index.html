Types of HTML Tags

Paired Tags

  1. Have an opening Tag.
  2. Have content / children tags.
  3. Have a closing Tag.
E.g. <p> </p> <ul> </ul>

Unaired Tags

  • Have an opening tag.
  • No content / children tags.
  • No closing tag.
E.g. <br> <hr>

Self-closing Tags

  • Only one tag that closes itself.
  • No content / children tags.
  • Require attributes.
E.g. <img src="img.png" />     <input type="text" />

Media Tags

Image Tag

Self-closing tag <img/>

Attributes:
  • src
  • alt
  • height/width in pixels

Audio Tag

Paired Tag <audio><audio/> Has one or multiple <source> tag

<audio> attributes:
  • controls
  • autoplay
  • loop
  • muted

<source> attributes:

  • src
  • type

Video Tag

Paired Tag <video><video/> Has one or multiple <source> tag

<video> attributes:
  • width
  • height
  • controls
  • autoplay
  • loop
  • muted

<source> attributes:

  • src
  • type

iFrames

<iframe> tag allows to embed an entire HTML document into any section of the current document.
