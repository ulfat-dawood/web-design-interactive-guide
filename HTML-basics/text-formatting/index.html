Semantic Tags

  • Communicate with the browser and add meaning to the text.
  • Imporoves accessibility.
<em> tells the browser this is emphasized text
<strong> tells the browser this is an important text

More Semantics

  • <blockquote> allows to quote text, and cite refference.
    if you plant the seeds from an apple, it will grow into a new variety.
  • <p> Shorter qoutes.
    To be, or not to be -shakespeare

  • <abbr> hover over this abbreviation GDP

  • <cite> to cite a piece of work
    The Fault In Our Stars by john green.

  • <dfn> Definitions
    Entropy is the measure of the unavailable energy in a closed thermodynamic system.

  • <address> Address information

    ulfat@icapm.dev Hyderabad, India.


  • <ins> <del> <s> indicate change to a text
    this WORD was inserted, this WORD was deleted and this WORD is no longer accurate.

Character Formatting

  • <sup> superscript: This is the 21st century.
  • <sub> subscript: CO2 levels today are relatively higher.
